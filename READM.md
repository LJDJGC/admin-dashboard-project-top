# Admin Dashboard

A responsive admin dashboard layout built with CSS Grid.

## Tech Stack
- HTML5
- CSS3 (Grid Layout)

## Features
- Responsive grid layout
- Sidebar navigation
- Project cards with auto-fit grid
- Clean, minimal design

## What I Learned
- CSS Grid fundamentals
- Nested grid containers
- Grid template areas
- Responsive card layouts using auto-fit
